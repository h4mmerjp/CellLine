# CellLine

ラベル付きグリッドエディタ。セルではなく**線（境界）にラベルを付ける**ことで、区切りそのものに意味を持たせられます。

## 機能

| 操作           | 方法                                                 |
| -------------- | ---------------------------------------------------- |
| セル編集       | ダブルクリック                                       |
| 範囲選択       | ドラッグ / Shift+クリック                            |
| セル結合・解除 | 範囲選択後、ツールバーの ⊞ / ⊟                       |
| 行・列の追加   | ツールバーのボタン、または ＋ ボタンで任意位置に挿入 |
| 行・列の削除   | ツールバーのボタン（末尾から削除）                   |
| サイズ変更     | ▼（列幅）/ ▶（行高さ）ハンドルをドラッグ             |
| ラベル編集     | ラベルをダブルクリック                               |
| ラベル並び替え | ⠿ を長押してドラッグ                                 |
| 保存           | ↓ 保存 → `cellline.json` をダウンロード              |
| 読込           | ↑ 読込 → `.json` ファイルを選択                      |

データはブラウザの `localStorage` に自動保存されます（ページをリロードしても状態が復元されます）。

## ユースケース

ラベルが「セル」ではなく「線（境界）」に付くため、**隣り合う複数セルにまたがる区間に名前を付けたい**場面で特に有効です。セルラベルで対応しようとすると結合が必要になるところを、線ラベルなら結合なしで区間の意味を表現できます。

### 時間・順序の区切り

- **週スケジュール** — 横線＝時間帯の境界（「9:00」「12:00」）、縦線＝曜日
- **工程表** — 横線＝フェーズ境界（「設計完了」「実装完了」）、縦線＝担当チーム
- **タイムボックス** — 横線＝セッション区切り、縦線＝並行トラック

### マトリクス評価

- **スキルマップ** — 横線＝スキル領域の境界、縦線＝メンバー
- **競合比較** — 横線＝評価軸の区切り（「機能」「価格」「サポート」）、縦線＝製品
- **RACI表** — 横線＝フェーズ境界、縦線＝役割（R/A/C/I）

### 空間・配置

- **座席表** — 横線＝列の境界、縦線＝席番号の境界、結合で机の島を表現
- **展示ブース配置** — 横線＝エリア区切り、縦線＝通路ライン
